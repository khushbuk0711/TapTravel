# Tap Travel - A Comprehensive Travel App

Tap Travel is an Android application that streamlines travel planning by integrating multiple features and APIs, including travel and weather data. It uses a map SDK for easy navigation and exploration of top destinations, with real-time information and personalized recommendations. The app includes user authentication, powered by Firebase, and manages wishlist data through Firestore. With its smooth interface and comprehensive functionality, Tap Travel offers a rich and interactive travel experience.
